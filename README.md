## 这个 App 到底在做什么

密码键盘的核心逻辑很简单：你只需记住**一个根密码**，它帮你记住所有其他的账号密码。

用起来也不复杂。把你的账号密码一一录入，下次需要登录某个 App 或网站时，唤出密码键盘，指纹验证，一键填充，搞定。

它有个独特的地方：**创新专利键盘形式**。说白了就是以输入法键盘的形态来填充密码，账号密码全程暗文显示，不会出现在系统剪贴板里，也不会被其他 App 读到。这对那些在公共场所经常要输入密码的人来说，安全感翻倍。

## 凭什么说它安全

"安全"这个词，软件厂商谁都会说。但密码键盘给出的是可验证的东西：

**加密算法层面**，采用 PBKDF2 + AES 双重加密，这是美国军方级别的标准，开发者本身也无法获取你的真实密码。

**本地优先**，你的根密码和解密密钥，永远只在你的设备上，永远不会上传到云服务器。就算服务器被黑，你的数据依然安全。

**支持完全离线使用**，可以在设置里直接禁掉网络权限，数据全程本地加密存储，断网用也没问题。

## 多设备同步怎么搞

有人可能会问：既然支持离线，那多设备同步怎么办？

密码键盘的方案是**加密云同步**：开启后，数据在上传前会先用 Bcrypt + AES 多层加密，传输全程走 HTTPS 协议。云端存的只是加密后的密文，没有你的根密码，任何人拿到那堆密文都是一堆乱码。

支持平台：**Android、iOS、Windows、Mac** 全覆盖。手机端激活会员后，电脑端免费使用。电脑端不存储数据，需要填密码时，打开手机 App 扫码登录，然后在电脑上选择账号填充即可。
